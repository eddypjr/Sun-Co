<h2>Description</h2>
<p>Discover <b>Sun Co.</b>, a serverless ecommerce web application that combines an intuitive design with a seamless user experience, enssuring a clutter-free shopping experience.</p>

<h3>Built with:</h3>

- React
- TypeScript
- Firebase
- Stripe API
- Styled Components
